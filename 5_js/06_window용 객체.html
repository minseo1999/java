window 객체

window 객체는 자바스크립트 최상위 객체이며 bom,dom으로 나뉨
- BOM (Brower Object Model) : location(주소관리창), screen(장비관련), navigator(브라우저 관련)
- DOM (Document Objct Model) : document 객체

- [window].open(["url"],"특성")
